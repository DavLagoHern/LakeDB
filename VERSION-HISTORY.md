# LakeDB version history

The README describes the current LakeDB experience. This page keeps the smaller changes between builds.

## 0.10

### 0.10.4

- Added a review table before importing DBeaver, SQLyog, JSON, CSV or SQL URL connections.
- Added row selection, editable names/users, shared or individual passwords and explicit duplicate handling.
- Added safe SQLyog password decoding with a clear fallback when the encryption is unsupported.
- Added the virtual **Unclassified** Home group for connections without folders.

[Release notes and downloads](https://github.com/DavLagoHern/LakeDB/releases/tag/v0.10.4)

### 0.10.3

- Introduced the split SQL/table workspace, schema-aware completion, alias and index guidance.
- Added compact/expanded formatting, typed result editing, cell-driven filters and visual schema management.
- Added compact two-row table controls, tab overflow navigation and restored window bounds.

[Release notes and downloads](https://github.com/DavLagoHern/LakeDB/releases/tag/v0.10.3)

## 0.9

### 0.9.3

- Added configurable complete-query exports up to 50 million rows.
- Added CSV, JSON, JSON Lines, Excel-compatible and SQL output with optional GZIP compression.

[Release notes and downloads](https://github.com/DavLagoHern/LakeDB/releases/tag/v0.9.3)

Older builds remain listed on the [GitHub Releases](https://github.com/DavLagoHern/LakeDB/releases) page.
