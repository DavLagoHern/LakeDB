# Colaborar con LakeDB

Gracias por ayudar a mejorar LakeDB. Este repositorio contiene las releases, la documentación y el espacio público de la comunidad; el código fuente de la aplicación se mantiene en privado.

## Formas de colaborar

- Proponer funciones y aportar casos de uso en [Discussions](https://github.com/DavLagoHern/LakeDB/discussions).
- Votar y comentar ideas existentes.
- Responder dudas de otros usuarios.
- Reportar fallos con pasos reproducibles.
- Proponer correcciones para el README y la Wiki.

## Reportes de errores

Utiliza el formulario de error e incluye la versión de LakeDB, macOS, arquitectura, motor de base de datos, resultado esperado y pasos para reproducirlo. Adjunta capturas o diagnóstico únicamente después de eliminar información sensible.

No publiques vulnerabilidades o posibles problemas de seguridad como una incidencia normal. Contacta primero con el mantenedor mediante su perfil de GitHub para acordar un canal privado.

## Pull requests

Se aceptan pull requests para documentación, recursos de comunidad y mejoras del repositorio público. Mantén cada cambio enfocado y explica qué problema resuelve. Las contribuciones al código de la aplicación no se gestionan desde este repositorio.

Al participar aceptas las [normas de la comunidad](CODE_OF_CONDUCT.md).
